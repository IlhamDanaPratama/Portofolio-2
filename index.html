Rancang Bangun Microwave Otomatis Berbasis IoT Menggunakan ESP8266

Tools: CorelDraw, Arduino IDE, Fritzing, Microsoft Excel

Description: Project ini menggunakan ESP8266 sebagai microcontroller, menggunakan heater AC 220V sebagai pemanas oven dengan kapasitas 1000W 2 buah. Sistem kerja dari prototype ini, setelah disetting suhu dan waktu maka heater akan aktif dengan bantuan relay sehingga memanaskan sesuai dengan setpoint. Suhu akan selalu terjaga dengan bantuan sensor DHT22 yang terpasang didalam oven, ketika setpoint waktu sudah habis maka oven akan berhenti secara otomatis dan buzzer akan menyala selama 5 detik. Proses tersebut akan dimonitoring menggunakan aplikasi Blynk IoT.

Menentukan Sistem Kerja Yang Divisualisasikan Menggunakan Flowchart: Pada tahap awal, saya membuat flowchart untuk menggambarkan alur kerja dari sistem microwave otomatis ini. Flowchart ini memetakan seluruh proses mulai dari input pengguna untuk pengaturan suhu dan waktu, aktivasi relay untuk menghidupkan heater, monitoring suhu secara real-time, hingga penghentian heater dan pemberitahuan menggunakan buzzer saat proses selesai. Visualisasi ini menjadi panduan utama dalam pengembangan perangkat lunak dan perangkat keras.

Membuat Script Prototype Menggunakan Arduino IDE: Tahap ini melibatkan pembuatan kode program menggunakan Arduino IDE untuk mengendalikan fungsi utama microwave otomatis. Kode ini meliputi pengaturan suhu dan waktu menggunakan input dari pengguna, pengontrolan relay untuk mengaktifkan heater, serta membaca data dari sensor DHT22 untuk memantau suhu di dalam microwave. Script juga mengelola komunikasi dengan aplikasi Blynk IoT untuk monitoring jarak jauh, serta mengendalikan buzzer sebagai notifikasi akhir proses.

Desain Rangkaian Elektronika Menggunakan Fritzing: Pada tahap ini, saya membuat diagram rangkaian elektronik menggunakan Fritzing. Rangkaian ini mencakup koneksi antara ESP8266 sebagai mikrokontroler, relay untuk mengontrol heater, sensor DHT22 untuk pengukuran suhu, serta komponen pendukung lainnya seperti sumber daya dan buzzer. Diagram ini digunakan untuk memastikan semua komponen terhubung dengan benar dan bekerja sesuai dengan desain sistem.

Desain Dashboard Menggunakan Cloud Blynk IoT Dan Blynk Mobile: Pada tahap ini, saya merancang dashboard di Blynk IoT yang berfungsi untuk monitoring dan kontrol microwave otomatis secara real-time. Dashboard ini memungkinkan pengguna untuk melihat suhu di dalam oven, mengatur timer, serta mengaktifkan atau menonaktifkan oven dari jarak jauh melalui aplikasi Blynk di perangkat mobile. Dua gambar yang ditampilkan menunjukkan tampilan dashboard di platform Blynk IoT dan aplikasi Blynk di smartphone, yang memberikan akses kontrol yang mudah dan praktis.

Tampilan Pengujian Microwave Otomatis Blynk Mobile: Tahap ini adalah pengujian akhir di mana microwave otomatis diuji secara langsung dengan menggunakan aplikasi Blynk di perangkat mobile. Saya memverifikasi bahwa seluruh fungsi mulai dari pengaturan suhu, waktu, aktivasi heater, hingga notifikasi buzzer berfungsi dengan baik. Selain itu, saya memastikan bahwa komunikasi antara perangkat dan aplikasi berjalan lancar tanpa ada keterlambatan. Gambar yang ditampilkan menunjukkan hasil pengujian ini dengan monitoring suhu secara real-time melalui aplikasi Blynk.
